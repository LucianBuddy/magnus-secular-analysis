```python
elif 现金转化率 > 80% 且 折旧/净利润 < 20%:
    oe_available = True
    oe_approx = np  # 高现金转化率+低折旧，OE≈净利润
else:

        检查应收账款变动+存货变动
        if 应收大幅增加: 标记"回款延迟"
        elif 存货大幅增加: 标记"备货周期，销售后可转化"
        else: 标记"需进一步核实"

    else:

# 不足60根K线时取实际根数
```

### 分支 A — 巴菲特框架（读取 buffett 参考文件）

**2-D 前读**：`frameworks/buffett/03-business-moat.md` → 5种护城河分类 + 趋势判断
**市场格局评分**：全球份额>60%或竞争对手<3家=主导/寡头(护城河上调一级) / 份额30-60%=领先 / <10%=碎片化/竞争激烈
**2-D 后读**：`frameworks/buffett/04-management-governance.md` → 三维管理评估

**客户集中度与地缘风险**：if 前3大客户收入占比>50%→高度集中；if 主要客户在海外→地缘政治风险存在

**知名投资机构持仓检查**（pass if 持有时间>5年：标记"长期持有，信任何极高"）

**管理层/大股东评估**（必做）：
1. 管理层诚信 — [是/否]
2. 大股东信用风险 — [低/中/高]（大股东有债务违约/破产重组记录则≥中）
3. 股权质押比例 — 若>50%标记风险

**杠杆质量分析**：
if 负债率 > 50% 且有息负债率 < 10%: 高负债但主要为经营性负债（应付账款/预收款），反映供应链议价能力，非财务风险
elif 负债率 > 50% 且有息负债率 > 30%: 高负债且主要为借款，财务杠杆风险需关注

**压力测试**（必做，假设营收下降50%）：
if 有息负债率 < 20% 且 现金/短期债务 > 2: 大概率能活
elif 有息负债率 > 50% 且 现金/短期债务 < 1: 风险高
else: 需进一步分析

**2-E 内在价值**：

*E1 — Owner Earnings*：从 `ss.oe_avg` 和 `ss.oe_pr` 读取（已预计算）。LLM 判断护城河强度确认倍数。

**若 Owner Earnings 不可用** → DDM替代（适用于高分红/公用事业/消费龙头等利率敏感型公司）。

两档输出：
DDM保守(g=0%)：最近3年平均股息 ÷ 无风险利率(10Y国债~2.5%)
DDM增长(g≈通胀率2%)：平均股息 ÷ (无风险利率-2%)

利率敏感性（公用事业/高分红股必做）：
DDM(2.5%) = 股息÷0.025
DDM(3.0%) = 股息÷0.03  # +50bp的影响
DDM(2.0%) = 股息÷0.02  # -50bp的影响
*E2 — PE估值*：`ttm_eps = 净利基数`；中枢 = median(ss.pe_h) 若数据不足则用行业均值 ±20%。

**前向PE**：
if 最新季度净利润同比 > 30%:
    前向PE = 市值 / (最新季度净利润 × 4)
    if 前向PE < TTM PE × 0.7: 标记"盈利快速增长，TTM PE滞后"
    elif 前向PE > TTM PE: 标记"季度盈利走弱或季节性淡季，全年盈利可能低于年化"，以TTM PE为主
else:
    无需计算前向PE
*E3 — PB估值*（重资产必做）：`median(ss.pb_h)` vs 当前 PB。
*E4 — 综合区间*：三条路径取交集。

**安全边际判定**（表）
| 估值vs中位 | 输出 | 持有操作 |
|------------|------|---------|
| 溢价>30% | 严重高估 | 观望→强烈不建议，卖出→自動判高估 |
| 溢价10-30% | 偏高 | 观望 |
| 折价10-20% | 合理偏低 | 可考虑买入 |
| 折价>20% | 低估，有安全边际 | 建议买入 |
| 折价>30%且Q5通过 | 显著低估 | 强烈建议买入 |

若MAX(PE,PB,DDM)/MIN(PE,PB,DDM) > 1.2 → 标记"估值方法分歧>20%"，分析分歧原因，取交集下沿。

**2-F 前读**：`frameworks/buffett/06-valuation-capital.md` → 安全边际分级
**2-F 持有结论**：买入/持有/卖出 + 推荐买入价（低于XX元，对应PE中位数×TTM EPS以下）+ EPS一致预期 `ss.eps`。

**买入价判断**：if 当前价 < 推荐买入价 → 输出"当前价已满足买入条件" / else → 输出"等待跌至推荐买入价以下"。

**盈利质量覆盖规则**：
if Q5=✅且Q8=❌: 输出"好公司不便宜"——质量极好但溢价>20%仍不建议买入，可上调安全边际至15%(非标准20%)
elif Q5=❌且Q8=✅: 输出"便宜没好货"——低估值但利润质量差，不建议买入
elif Q5=No且Q8=Yes: 盈利质量差覆盖估值优势，结论偏谨慎
else: 正常买入

**卖出条件逐项检查**（读 buffett `frameworks/buffett/07-risk-behavior.md`）：
1. 价格严重高估？[是/否+依据]
2. 护城河被破坏？[是/否+依据]
3. 管理层诚信问题？[是/否+依据]（一票否决，是则立刻卖出）
4. 出现更好的机会？[是/否+依据]

## Step 4：输出报告

### A. 报告结构

```

```

### B. 决策模块

默认按"未持仓"输出。分支A：低估→买入/合理→观望/高估→卖出（配≤3观察变量）。分支B：相对分位XX% → 持有/减仓/观望。

### B2. 监控指标（必出）

```
每季度检查：
  ROE趋势、毛利率变化、经营现金流方向、客户集中度变化
触发重新评估的信号：
```
