# Financial Metrics — Summary

核心指标：
- **ROE**（净资产收益率）：长期 > 15% 为好，但要排除高杠杆推高的情况。
- **Owner Earnings**（所有者盈余）：净利润 + 折旧摊销 - 维持性资本开支。这是巴菲特最看重的单指标。
- **现金转化率**：经营现金流 / 净利润 > 0.8 为合格，> 1.2 为优秀。
- **毛利率**：护城河的量化表现。连续3年下降 > 5% 是红灯。
- **有息负债率**：< 30% 为安全，> 50% 需警惕（金融/银行除外）。

杜邦拆分：ROE = 净利率 × 资产周转率 × 权益乘数
- 高杠杆推高的 ROE 不如高利润率推高的 ROE 可持续
